# Search Extension

This Chrome extension extracts search queries from Google search results and injects sarcastic responses into the page.

## Features

- Extracts search queries from Google.
- Injects humorous, sarcastic responses into the search results.

## Installation

1. Clone the repository or download the ZIP file.
2. Navigate to `chrome://extensions/` in your Chrome browser.
3. Enable "Developer mode" in the top right corner.
4. Click on "Load unpacked" and select the `search-extension` directory.

## Usage

1. Perform a search on Google.
2. Click on the extension icon in the Chrome toolbar.
3. Enjoy the sarcastic responses injected into your search results!
